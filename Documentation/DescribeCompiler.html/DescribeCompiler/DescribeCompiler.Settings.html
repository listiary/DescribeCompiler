DescribeCompiler
#Settings

Those are hardcoded settings that can be used to change the behaviour of the compiler.


const bool PARSE_DS_ONLY = true
Wether to parse only .ds files, or all the files in a directory.


const bool PARSE_TOP_DIRECTORY_ONLY = false
Wether to parse files in all subdirectories or only in the top directory


const string COMPILER_NAME = "Describe Compiler v0.9.2"
A string representing the full name of the compiler (with the version)


const string COMPILER_VER = "0.9.2"
A string representing the compiler version


const GrammarName DEFAULT_GRAMMAR = GrammarName.Decorators
The default grammar file to be loaded with this version of the compiler.
The grammar is usually left alone, unless there is a very good reason to be changed, which, on the other hand, makes this the de facto grammar that is used.
