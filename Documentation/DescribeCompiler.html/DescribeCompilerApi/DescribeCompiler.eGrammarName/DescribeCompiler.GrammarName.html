DescribeCompiler.GrammarName

The GrammarName enum is used to describe the version of the grammar that is used with the parser engine.
It has 5 values:


Basic
Corresponds to grammar version 0.6.
This version incorporates only basic syntax.

Tags
Corresponds to grammar version 0.7.
This version introduces the concept of tagging.

Links
Corresponds to grammar version 0.8.
This version introduces the concept of links.

Decorators
Corresponds to grammar version 0.9.
This version introduces the concept of working with decorators.

Official
Corresponds to grammar version 1.0.
This version is not available yet!
