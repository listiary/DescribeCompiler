DescribeCompilerCLI


DescribeCompilerCLI is the CLI interface for the describe compiler, which is used to work with files and folders.
There are 4 distinct modules in the CLI app:

DescribeCompilerCLI.Messages
Used to set up the console and to display various messages

DescribeCompilerCLI.Arguments
The methods that are used to read the command line arguments are in this module

DescribeCompilerCLI.Datnik
The Datnik structure is used to store data aquired from the command line, via the methods in the Arguments module

DescribeCompilerCLI.Main
The methods of the CLI app that do the actual work are here
