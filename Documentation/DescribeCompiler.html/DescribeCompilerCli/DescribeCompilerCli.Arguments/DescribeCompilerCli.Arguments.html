DescribeCompilerCLI.Arguments


This module of the CLI app contains all the methods used to read the invocation arguments from the command line that started the exe.
If successful, the information is stored in the Datnik structure:


static bool readInputArgument(string arg, int argindex)
Attempt to read the input argument and set the flag indicating wether it is a file or a folder
The index of the argument is needed for logging purposes


static bool readOutputArgument(string arg, int argindex)
Attempt to read the output argument and set the flag indicating wether it is a file or a folder
The index of the argument is needed for logging purposes


static bool readVerbosityArgument(string arg, int argindex)
Attempt to read the verbosity argument that indicates the verbosity level to be used with the parser
The index of the argument is needed for logging purposes


static bool readTemplateArgument(string arg, int argindex)
Attempt to read the template argument that indicates which Translator and which template set are to be used with the parser
The index of the argument is needed for logging purposes
