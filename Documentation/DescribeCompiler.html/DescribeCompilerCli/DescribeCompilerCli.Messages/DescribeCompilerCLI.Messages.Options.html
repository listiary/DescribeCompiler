static bool ONE_BASED_ARG_INDEX
Wether we are using one based index for error reporting.
For example, if there is an error in the first cmd argument, if ONE_BASED_ARG_INDEX is true, we will get something like "Error at argument 1". Otherwise we will get "Error at argument 0".
