#PresettingMethods


PRESETTING METHODS:

presetConsole()
Set console dimensions and colors

printLogo3(ConsoleColor logoColor)
Output the ASCII art logo with 3 dashes for shadow to the console

printLogo3()
Output the ASCII art logo with 3 dashes for shadow to the console, using the default color for text

printLogo2(ConsoleColor logoColor)
Output the ASCII art logo with 2 dashes for shadow to the console.

printLogo2()
Output the ASCII art logo with 2 dashes for shadow to the console, using the default color for text

printLogo3Bicolor(ConsoleColor colorA, ConsoleColor colorB)
Output the ASCII art logo with 2 dashes for shadow to the console, in 2 alternating colors.

printLogo3Bicolor()
Output the ASCII art logo with 2 dashes for shadow to the console, in 2 alternating colors, using the default color for text and the default color for info

PRESETTING METHODS - THEMES:

SetDefaultTheme()
Set default color theme

SetVioletTheme()
Set white/violet color theme

SetCyanTheme()
Set cyan/violet color theme

SetLightBlueTheme()
Set light blue/white color theme

SetDarkBlueTheme()
Set dark blue/dark gray color theme
