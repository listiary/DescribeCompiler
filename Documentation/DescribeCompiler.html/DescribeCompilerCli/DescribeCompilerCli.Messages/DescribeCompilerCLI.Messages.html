DescribeCompilerCLI.Messages


There are 3 distinct regions (for a lack or a better word) here:

OPTIONS
Settable options for the Messages module.

PRESET CONSOLE METHODS
Methods used to change the console appearance like size or colors at the start of the CLI program execution.
Also, methods for writing te logos to the console.

ERROR HANDLING METHODS
Methods that output error messages to the console and block execution by waiting for user input (press any key to exit).

LOG METHODS
Methods that are used to output specific kind of messages to the console.
Those methods are passed to the compiler as log handlers.
