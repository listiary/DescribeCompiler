DescribeCompilerDb.CliCommand

The CliCommand enum is used to describe the command that is to be executed.
It has 6 values:


Update_Database
Indicates that we are doing database update command.
DescribeCompilerDb updateDatabase "path/saved_databases/"
DescribeCompilerDb updateDb "path/saved_databases/"
DescribeCompilerDb update-database "path/saved_databases/"
DescribeCompilerDb update-db "path/saved_databases/"

Update_JsonFile
Indicates that we are doing ready json update command.
DescribeCompilerDb updateJson "path/ready_data.json"
DescribeCompilerDb update-json "path/ready_data.json"

Produce_UploadSql
Indicates that we are doing produce upload sql command.
produce-uploadqueries "path/input directory/" "path/output.sql"
produce-uploadsql "path/input directory/" "path/output.sql"
produceUploadQueries "path/input directory/" "path/output.sql"
produceUploadSql "path/input directory/" "path/output.sql"

Produce_JsonFile
Indicates that we are doing produce json command.
produce-json "path/input directory/" "path/ready.json"
produceJson "path/input directory/" "path/ready.json"

ProduceUpdate_JsonFile
Indicates that we are doing produce-update json command.
produceupdate-json "path/input directory/" "path/ready.sql"
produceUpdateJson "path/input directory/" "path/ready.sql"

Download_Database
Indicates that we are doing database download command.
DescribeCompilerDb downloadDatabase "path/saved_databases/"
DescribeCompilerDb downloadDb "path/saved_databases/"
DescribeCompilerDb download-database "path/saved_databases/"
DescribeCompilerDb download-db "path/saved_databases/"

Help
Indicates that we are doing help command which outputs usage data to the console.
DescribeCompilerDb help

Undefined
Indicates that we don't have a command and need to show error message and exit the app.
DescribeCompilerDb help"
