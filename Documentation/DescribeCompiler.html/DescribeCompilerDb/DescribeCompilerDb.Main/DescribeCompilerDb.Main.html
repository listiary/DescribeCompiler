DescribeCompilerDb.Main


This module of the CLI app contains all the methods used to do the actual work


static bool Update_Database()
Download database, produce ready data file (json) and update it.
DownloadDatabase(...); ProduceReadyData(); UpdateReadyData();


static bool Update_ReadyData()
Update the compiled .json file that is used to display the website.


static bool Produce_ReadyData()
Compile a directory of ds source files to json ready to be used in the website.


static bool Produce_UploadSql()
Compile a directory of ds source files to sql queries that will insert those files in the database.


static bool ProduceUpdate_ReadyData()
Compile a directory of ds source files to json ready to be used in the website, then update the compiled .json file that is used to display the website.


static bool Download_Database(out string outdir)
Download the website database in a separate, timestamped folder. That folder is created in the output directory.
outdir - The path of the created timestamped directory
